\subsection{Building the website}\label{building-the-website}

Unicode \texttt{\textbackslash{}u200B} for zero width space, e.g., in
\texttt{\_quarto.yml} I used it for an item with only an icon in the
navigation bar. Note that I tried HTML's \texttt{\&\#8203;} but it
doesn't work.

Extensions for adding icons such as google scholar and 500px
\faIcon{500px} (see the website's footer):

\begin{Shaded}
\begin{Highlighting}[]
\BuiltInTok{.}
\ExtensionTok{├──}\NormalTok{ quarto{-}ext}
\ExtensionTok{│}\NormalTok{   └── fontawesome}
\ExtensionTok{└──}\NormalTok{ schochastics}
    \ExtensionTok{└──}\NormalTok{ academicons}
\end{Highlighting}
\end{Shaded}

For publishing a Quarto website, I recommend using the second option in
the \href{https://quarto.org/docs/publishing/github-pages.html}{official
documentation}. After set up, simply call:

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{quarto}\NormalTok{ publish gh{-}pages }\AttributeTok{{-}{-}no{-}prompt}
\end{Highlighting}
\end{Shaded}
